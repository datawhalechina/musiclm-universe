    音乐生成模型原理、优化与项目实战

    MusicLM-Universe：系统化讲解音乐生成语言模型（Music Language Model）的开源教程项目，兼顾理论基础、实践部署与进阶优化，助力开发者从入门到精通音乐生成技术

    🎧 音乐生成模型体验指南

    本教程首先带你体验当前主流的音乐生成模型，通过网页即可快速生成音乐。

    1st: Suno

    • 网址：https://suno.com
    • 简介：Suno 是目前最流行的音乐生成平台之一，可以直接生成完整歌曲（人声 + 编曲 + 歌词）

    特点

    • 支持歌词 + 风格控制
    • 自动生成结构（verse、chorus）
    • 很适合完整歌曲创作

    🎯Prompt

    A dreamy pop song with female vocals, about loneliness in a futuristic city, slow tempo, emotional, with piano and soft synth pads

    一首梦幻流行歌曲，女声演唱，主题是未来城市中的孤独感，慢节奏，情绪化，包含钢琴和柔和合成器

    技巧：

    • 加上 “in the style of …” 可以增强风格
    • 指定 BPM 或者 key 会更专业

    2nd: Riffusion

    • 网址：https://www.riffusion.com
    • 简介：Riffusion 是一个偏实验性的音乐生成工具，主打快速生成 + 免费使用

    特点

    • 生成速度快
    • 更适合 loop、灵感片段

    🎯 Prompt

    lofi hip hop beat, chill vibe, vinyl crackle, jazzy chords, 70 bpm

    lofi 嘻哈节奏，放松氛围，黑胶噪音，爵士和弦，70 BPM

    技巧：

    • 关键词越短越好
    • 多用风格标签（lofi、trap、ambient）

    3rd: Udio

    • 网址：https://udio.com
    • 简介：Udio 是近年来非常强的音乐生成工具，和 Suno 类似，但在某些风格上更自然

    特点

    • 强人声表现
    • 结构更自然
    • 适合 storytelling 音乐

    🎯 Prompt

    cinematic orchestral track, epic build-up, trailer music, powerful drums, emotional strings

    电影感管弦乐，史诗级推进，预告片音乐，强力鼓点，情绪弦乐

    4th: MusicGen（Meta 开源模型）

    • 体验入口：https://www.musicgen.app
    • 简介：MusicGen 是 Meta 推出的开源音乐生成模型，属于 AudioCraft 体系

    特点

    • 开源
    • 可控性强
    • 适合研究和开发

    🎯 Prompt

    funky disco groove with bass guitar, drums, and electric piano, upbeat and danceable

    放克迪斯科律动，包含贝斯、鼓和电钢琴，节奏欢快，适合跳舞

    技巧：

    • 描述乐器组合非常关键
    • 多用节奏描述

    🎯 Prompt 编写通用模板

    [风格] + [情绪] + [乐器] + [节奏/BPM] + [场景/用途]

    示例：

    jazz, relaxing, saxophone and piano, slow tempo, late night cafe atmosphere
